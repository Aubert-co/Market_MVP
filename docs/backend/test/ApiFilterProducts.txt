Funcionalidade: Filtrar produtos por: categoria,name,maxPrice,minPrice
  Como um usuário do sistema
  Quero buscar produtos filtrando por categoria
  Para visualizar apenas os produtos relevantes ao meu interesse

  Cenário: Buscar produtos pela categoria "Eletrônicos"
    Dado que existem produtos cadastrados nas categorias "Eletrônicos" , "Moda" , "Acessorios,"Informatica"
    E a API de filtrar produtos está disponível
    Quando o usuário envia uma requisição POST para "/product/filter" com body {category:'test'}
    Então a API deve retornar apenas os produtos da categoria "Eletrônicos"
    E o status da resposta deve ser 200
  

  Cenário: Buscar produtos pelo nome
    Dado que existem produtos cadastrados nas categorias "Eletrônicos" , "Moda" , "Acessorios,"Informatica"
    E a API de filtrar produtos está disponível
    Quando o usuário envia uma requisição POST para "/product/filter" com name igual a 4 letras 
    Então a API deve retornar um error "Invalid name format"
    E o status da resposta deve ser 400
